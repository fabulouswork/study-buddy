# Exercise 6: Special Matrices and Transformations

## Problem

Match each of the following matrices with the **transformation it represents** (in 2D):

$$P = \begin{pmatrix} 1 & 0 \\ 0 & 0 \end{pmatrix}$$

$$R = \begin{pmatrix} 0 & -1 \\ 1 & 0 \end{pmatrix}$$

$$S = \begin{pmatrix} 1 & 0 \\ 0 & -1 \end{pmatrix}$$

$$I = \begin{pmatrix} 1 & 0 \\ 0 & 1 \end{pmatrix}$$

**Choose from:**
- (a) Identity (do nothing)
- (b) Reflection across the $x$-axis
- (c) Rotation by 90° counterclockwise
- (d) Projection onto the $x$-axis

Then, for each matrix, describe what happens to the basis vectors $\mathbf{e}_1 = \begin{pmatrix} 1 \\ 0 \end{pmatrix}$ and $\mathbf{e}_2 = \begin{pmatrix} 0 \\ 1 \end{pmatrix}$.

---

**Hint (if needed):** Try applying each matrix to $\mathbf{e}_1$ and $\mathbf{e}_2$. The results are literally the columns of the matrix!
