# Exercise: Substitution Mini-Quiz (Ex007)

Solve using substitution where appropriate.

## Instructions

- Show your substitutions.
- Include constants correctly.
- Use absolute values in logs when needed.

1. Note: Since \(x^2+1>0\), absolute value is optional in the final simplified form.
