# Study Buddy — Telegram Commands

Send these to Aura on Telegram to interact with study-buddy:

## General

| Command | Action |
|---------|--------|
| `/study status` | Show current progress, last studied, next steps |
| `/study progress` | Full progress dashboard |

## Learning

| Command | Action |
|---------|--------|
| `/study teach <subtopic>` | Teach a subtopic (e.g., `/study teach tensor-products`) |
| `/study quiz` | Take a quiz on the current topic |
| `/study quiz <subtopic>` | Quiz on a specific subtopic |
| `/study revise` | Revise a past exercise (random revisit) |
| `/study ready` | Self-report ready for next topic (after 100% quiz) |

## Exercises

| Command | Action |
|---------|--------|
| `/study exercise new` | Generate and send a new exercise |
| `/study exercise <NNN>` | Show a specific past exercise (e.g., `/study exercise 007`) |
| `/study answer <text>` | Submit your answer to the current exercise |

## Progress

| Command | Action |
|---------|--------|
| `/study next` | Move to the next curriculum topic (only if readiness gate passed) |
| `/study summary` | Full curriculum overview with ETA |
